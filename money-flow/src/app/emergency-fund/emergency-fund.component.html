Emergency Fund

After you start to gain some disposable income, make sure to set all ${{netExpenses}} of it aside into an emergency fund.

After you start to gain some disposable income, make sure to set $1000 of it each month into an emergency fund.

After a couple months, you will have enough to be able to cover any accidents or unexpected costs easily.



Non-Essential Bills

Non-Essential Bills include things like paying for cable, phone, internet, streaming services, and so on.

These are things that are not really necessary but can be accommodated for once enough disposable income is saved.
